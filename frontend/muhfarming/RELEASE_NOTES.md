# muhfarming Android App - v1.0.0

## 📱 Installation Instructions

### For Android Users (Android 11+)

1. **Download the APK**
   - Download `app-release.apk` from this release

2. **Enable Installation from Unknown Sources**
   - Go to **Settings** > **Security** (or **Privacy**)
   - Enable **"Install unknown apps"** for your browser or file manager
   - On some devices: **Settings** > **Apps** > **Special app access** > **Install unknown apps**

3. **Install the App**
   - Open your Downloads folder
   - Tap on `app-release.apk`
   - Follow the installation prompts
   - Tap **"Install"**

4. **Launch the App**
   - Find "muhfarming" in your app drawer
   - Tap to open

### ⚠️ Security Note

This APK is properly signed and safe to install. Android may still show a warning because it's not from the Play Store. This is normal for direct APK distribution.

## ✨ Features

### 🌤️ Weather Dashboard
- Real-time weather data for 5 Cameroon locations:
  - Yaoundé (Centre)
  - Douala (Littoral)
  - Bamenda (North-West)
  - Garoua (North)
  - Ngaoundéré (Adamawa)
- Current conditions, hourly forecast, and daily forecasts
- Temperature, humidity, wind speed, precipitation
- Soil temperature and moisture data
- Weather icons for all conditions
- Data powered by Open-Meteo API

### 🐛 Pest Management
- Comprehensive pest information for Cameroon farmers
- **Diseases covered:**
  - Downy Mildew (_Peronospora parasitica_)
  - Black Rot (_Xanthomonas campestris_)
  - Leaf Spot (_Cercospora, Alternaria_)
  - Damping-Off (_Pythium, Rhizoctonia_)
- Regional pest patterns:
  - _Tuta absoluta_ and bacterial wilt (West/Littoral)
  - Fall armyworm (Centre/Adamawa/North)
  - Fungal diseases (humid South)
  - Viruses and mites (drier North)
- Severity levels and seasonal information

### ⚙️ Settings
- Location selection for weather forecasts
- Clean, modern Material 3 design
- SAP Fiori color scheme
